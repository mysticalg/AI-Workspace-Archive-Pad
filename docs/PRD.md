# AI Workspace Archive PRD

This markdown copy mirrors the PDF product requirements document supplied with the repository request.

## Core product

- Manifest V3 Chrome extension
- Local-first archive in IndexedDB
- React popup, side panel, and options experiences
- Capture from ChatGPT, Claude, Gemini, and later Perplexity
- Project organization, tags, notes, snippets, search, and exports

## MVP scope implemented in this repository

- Save visible chat, selected range, and last exchange from supported pages
- Project management with local storage
- Search across local archive
- Export to Markdown, JSON, PDF, DOCX, and project ZIP
- Options page for storage defaults, platform controls, import, and deletion
- Backend and website scaffolding for later sync, billing, and launch work

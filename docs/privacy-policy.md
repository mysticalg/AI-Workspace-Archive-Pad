# Privacy Policy

AI Workspace Archive is designed to capture AI workspace content only when the user explicitly triggers a save action.

## Data handling

- Default mode stores records locally on the user device.
- The extension does not log unrelated browsing history.
- The extension does not capture keystrokes.
- The extension does not transmit content to a backend unless sync-enabled features are explicitly wired and enabled.

## Deletion

- Users can wipe local archive data from the options page.
- Archived items can be deleted individually from the side panel.

## Permissions

- Host access is limited to supported AI domains.
- Storage access is used for project, archive, snippet, and settings persistence.
